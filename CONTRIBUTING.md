# Contributing

This repository is a template for building your own GitHub profile README -
it's not a piece of software with a shared codebase, so "contributing" here
mostly means using it for yourself.

## If you want to use this template

1. Don't fork it as-is and leave it unedited - a profile README full of
   someone else's name, links, and stats isn't useful to you or them.
2. Follow `SETUP.md` to swap in your own username, links, and content.
3. Only keep the sections that are true for you (see the "optional" notes
   in `SETUP.md` for LeetCode, Discord badges, and donations) - a section
   showing all zeros or fake data looks worse than not having it at all.

## If you find a bug or a broken widget URL

Third-party services (stats cards, streak counters, etc.) occasionally
change or go down. If you notice something in this template pointing at a
dead service, feel free to open an issue describing what's broken.
