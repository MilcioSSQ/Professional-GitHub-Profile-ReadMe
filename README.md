# GitHub Profile README Template

A clean, working template for a GitHub profile README — stats, snake
animation, tech stack badges, Discord badges, LeetCode card, and more.

## What's included

- Typing animation header
- About Me section with a code block
- Tech stack badges — languages, frameworks, databases, DevOps, tools
- GitHub Contribution Snake (auto-updated via GitHub Actions every 12 h)
- GitHub Stats — commits, stars, PRs, top languages
- Current Streak counter
- GitHub Trophies
- Pinned project cards
- Activity graph
- LeetCode stats card
- Connect / social buttons — Discord, GitHub, Ko-fi, Twitch, Twitter
- Discord badge row
- Detailed profile summary cards
- Random dev quote

## How to use

> **The file to copy is [`PROFILE_README.md`](PROFILE_README.md)** —
> not this file. This file (`README.md`) is just the template description
> you're reading right now.

1. Create a repository named **exactly like your GitHub username**
2. Copy `PROFILE_README.md` into it and rename it to `README.md`
3. Copy `.github/workflows/snake.yml` into the same repo
4. Follow **[SETUP.md](SETUP.md)** to replace usernames, links, and personal info
5. Run the snake workflow once from the Actions tab
6. Push — your profile README appears at `github.com/your-username`
