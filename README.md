# GitHub Profile README Template

A clean, working template for a GitHub profile README — stats, snake animation, tech stack badges, Discord badges, LeetCode card, and more.

![Last updated](https://img.shields.io/badge/updated-July%202026-blue?style=flat-square)
![License](https://img.shields.io/badge/license-MIT-green?style=flat-square)

---

## What's included

- **Typing animation** header
- **About Me** section with a code block
- **Tech stack badges** — languages, frameworks, databases, DevOps, tools
- **GitHub Contribution Snake** — auto-updated via GitHub Actions every 12 hours
- **GitHub Stats** — commits, stars, PRs, top languages
- **Current Streak** counter
- **GitHub Trophies**
- **Pinned project cards**
- **Activity graph**
- **LeetCode stats card**
- **Connect / social buttons** — Discord, GitHub, Ko-fi, Twitch, Twitter
- **Discord badge row**
- **Detailed profile summary cards**
- **Random dev quote**

---

## How to use

See **[SETUP.md](SETUP.md)** for step-by-step instructions.

The short version:

1. Create a repository named exactly like your GitHub username
2. Copy the files from this template into it
3. Follow `SETUP.md` to replace usernames, links, and personal info
4. Run the snake workflow once from the Actions tab
5. Push — your profile README appears at `github.com/your-username`
